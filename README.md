# DigiTools 🛠️

A modern and responsive digital tools e-commerce platform built with React, featuring a sleek UI , shopping cart functionality, and pricing tiers.

## Tech Stack

- **React** - Frontend framework
- **Tailwind CSS** - Styling
- **DaisyUI** - UI components
- **Vite** - Build tool

## Features

- 🛒 **Shopping Cart** - Add/remove products with real-time cart updates and toast notifications
- 📱 **Responsive Design** - Fully responsive layout that works on all devices
- 💳 **Pricing Plans** - Multiple pricing tiers with feature comparison
